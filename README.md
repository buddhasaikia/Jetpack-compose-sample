# Jetpack Compose Sample

Things covered-

* Row
* Column
* Box
* Text modification
* Text selection
* BaselineShift (Superscript/Subscript)
* Custom View (Expandable card)
* TextFields (BasicTextField, TextField, OutlinedTextField)
* Coil image loading library
* Password TextField
